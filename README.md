La aplicación "PorcenCal Beta" es una calculadora de porcentajes desarrollada para dispositivos Android por Magic Code Inc. Está diseñada para realizar cálculos de porcentaje de manera rápida y sencilla, enfocándose en incrementos y descuentos porcentuales.

### Funcionalidades principales:
- Entrada de datos: El usuario ingresa un "Valor Inicial" (número decimal) y un "Porcentaje %" (también decimal).
- Selección de formato: Un spinner permite elegir la precisión decimal de los resultados (0, 0.00, 0.000, 0.0000 decimales).
- Cálculos realizados:
  - Incremento fijo del 5%: Muestra el valor inicial más un 5% de incremento.
  - Decremento fijo del 5%: Muestra el valor inicial menos un 5% de decremento.
  - Diferencia porcentual: Calcula y muestra únicamente el monto correspondiente al porcentaje ingresado sobre el valor inicial (sin sumar/restar).
  - Resultado final de incremento: Valor inicial + el porcentaje aplicado.
  - Resultado final de disminución: Valor inicial - el porcentaje aplicado.
- Interfaz: Pantalla limpia con una tarjeta central blanca sobre fondo gris claro, incluyendo una barra de herramientas superior y anuncios de banner inferior (usando AdMob).
- Menú adicional: Acceso a "Acerca de" (diálogo con información de la app), política de privacidad (enlace web) y enlace a la versión Pro en Google Play.

### Características técnicas:
- Plataforma: Android nativa, construida con Gradle.
- Lenguaje: Java.
- Monetización: Incluye anuncios de banner.
- Compatibilidad: Soporta edge-to-edge display y ajustes de insets para barras del sistema.
- Versión: Beta, con referencias a una versión Pro más avanzada.

Esta app es útil para cálculos financieros, comerciales o educativos donde se necesiten porcentajes variables, con ejemplos fijos del 5% para referencia rápida. Es una herramienta versátil y amigable, según su descripción interna.
